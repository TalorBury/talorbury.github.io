Git基本用法

2016/3/4 posted in  Git

1.Git安装

(1)Linux 安装 使用sudo apt-get install git
(2)Mac 安装 在AppStore中安装Xcode即可
(3)安装完成之后还需设置用户信息：
        git config --global user.name "Your name"
        git config --global user.email "Your email"

2.创建仓库

在需要创建仓库的文件夹中使用命令
        git init

3.添加文件到仓库

        git add readme.txt
        git add --ignore-removal .
        
        (只是将文件添加到了暂存区)

4.将文件提交到仓库

        git commit -m "Commit Description"

5.查看仓库状态

        git status

6.查看文件的修改部分

        git diff readme.txt

7.查看git历史记录

        //--pretty=oneline 显示的是commit id 版本号
        git log (--pretty=oneline)

8.回退到任意版本

        //HEAD表示当前版本
        git reset --hard HEAD~(n)
        //输入commit id 前几位即可
        git reset --hard commit id

9.查看git每一次的命令

        //可以查到用于回退任意版本的commit id
        git reflog

10.撤销修改

        //丢弃工作区的修改
        git checkout -- readme.txt
        //把暂存的修改回退到工作区
        git reset HEAD readme.txt

11.删除文件

        //删除工作区的文件
        rm readme.txt
        //删除版本库中的文件
        git rm readme.txt
        //提交删除
        git commit -m "delete readme.txt"
        //用版本库里的版本替换工作区的版本
        git checkout -- readme.txt

12.远程仓库

    （1）创建SSH Key
        ssh-keygen -t rsa -C "Your email"
    （2）Add SSH Key
        用户主目录中.ssh目录中有私匙id_rsa和公匙id_rsa.pub
        .ssh目录是隐藏文件，使用ls -ah可以查看所有文件
        远程的Key中填写id_rsa.pub文件中的内容
    （3）在GitHub中创建Repository
    （4）建立本地仓库与远程仓库的关联
        git remote add origin https://github.com/UserName/Repository.git
        (git remote rm origin  去除本地仓库与远程仓库的关联)
    （5）将本地仓库的内容推送到远程库中
        git push -u origin master
    （6）将远程库克隆到本地
        git clone https://github.com/UserName/Repository.git
