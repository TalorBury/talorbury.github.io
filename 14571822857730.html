Mac使用技巧

2016/3/5 posted in  Mac

1.如何输入Emoji

    Ctrl + Command + 空格键

2.窗口操作

    (1)关闭窗口
        Command + w
    (2)最小化窗口
        Command + m
    (3)隐藏窗口
        Command + h

3.睡眠

    Control + Shift + Power

4.外接屏幕时Dock的位置

    想在那个屏幕使用Dock，就将鼠标移动到最底部

5.使用open命令开启多个相同应用

    open -n /Applications/QQ.app/
